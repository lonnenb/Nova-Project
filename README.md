# Nova-Project
 Participants: Rosa, Fynn, Moritz and Benjamin

## Install dependencies

### Backend
```
cd app/backend
pip install -r requirements.txt
```

### Frontend
```
cd app/frontend
npm install
```

## Starting the Application

### Backend
```
cd app/backend
uvicorn main:app --reload
```

### Frontend
```
cd app/frontend
npm run dev
```
